# 🧠 O QUE É BACKEND?

Backend é a parte responsável por:

- Processar regras de negócio
- Validar dados
- Acessar banco de dados
- Retornar respostas

---

## ⚙️ Exemplo real (Login)

1. Usuário digita login
2. Frontend envia dados
3. Backend valida
4. Consulta banco
5. Retorna resposta
