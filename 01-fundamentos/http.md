# HTTP (HyperText Transfer Protocol)

## 📌 O que é?
Protocolo de comunicação usado entre cliente e servidor.

## 🔄 Como funciona?
- Cliente faz uma requisição
- Servidor responde

## 📬 Métodos principais
- GET → buscar dados
- POST → enviar dados
- PUT → atualizar
- DELETE → remover

## 📊 Status Code
- 200 → OK
- 404 → Não encontrado
- 500 → Erro no servidor
